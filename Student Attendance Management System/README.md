# 📌 Attendance Management System

## 📖 About the Project
The Attendance Management System is a simple and efficient web-based project developed using Codia.ai workspace. It helps to manage and track student/employee attendance digitally, reducing manual work and errors.

---

## 🚀 Features
- Add student/employee details  
- Mark daily attendance (Present/Absent)  
- View attendance records  
- Search and manage data easily  
- Simple and user-friendly interface  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- Codia.ai Workspace
